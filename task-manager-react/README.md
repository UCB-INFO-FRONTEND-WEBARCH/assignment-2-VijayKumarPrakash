# Task Manager React App

This is a React version of the Task Manager from Assignment 1.

## Project Structure

```
task-manager-react/
├── index.html
├── package.json
├── vite.config.js
└── src/
    ├── main.jsx
    ├── App.jsx
    ├── App.css
    ├── assets/
    │   ├── calendar_icon.png
    │   ├── check_icon.png
    │   ├── inbox_icon.png
    │   ├── menu_icon.png
    │   ├── search_icon.png
    │   └── upcoming_icon.png
    └── components/
        ├── TaskForm.jsx
        ├── TaskList.jsx
        ├── TaskItem.jsx
        └── TaskCounter.jsx
```

## Features

- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Filter tasks (All, Active, Completed)
- Task counter showing completed/total tasks
- Responsive design
- Clean, user-friendly interface

## Development

To run the development server:

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

To build for production:

```bash
npm run build
```

## Technologies Used

- React 18
- Vite
- CSS3
